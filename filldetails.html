ABOUT US

Seeking-CRM is a free and open source program to help small businesses with sales and services. It has a web interface and works from any modern browser. It is currently in the testing phase (beta version). Seeking-CRM is software that needs to be installed on the server. Don't worry about the word "server" - you don't have to have a real expensive server these days. You can completely get by with a virtual server (VPS-server), the services for which are provided by many hosting companies, and it will cost no more than regular website hosting
